# MaxyStyles - Fashion E-commerce Platform

MaxyStyles is a modern fashion e-commerce web application built with Next.js, MongoDB, Tailwind CSS, and shadcn/ui. It allows brand owners to upload clothing designs, set prices, manage variations, and interact with customers.

## 🚀 Features

- **Modern Tech Stack**: Next.js 15, TypeScript, Tailwind CSS, MongoDB
- **Beautiful UI**: Built with shadcn/ui components
- **Responsive Design**: Mobile-first approach
- **Product Management**: Add, edit, and manage clothing items
- **Category System**: Organize products by categories (Dresses, Tops, Bottoms, etc.)
- **Admin Dashboard**: Manage products, orders, and customers
- **Customer Features**: Browse products, shopping cart functionality
- **MongoDB Integration**: Robust data models for products, users, and orders

## 🛠️ Tech Stack

- **Frontend**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS, shadcn/ui
- **Backend**: Node.js, Next.js API Routes
- **Database**: MongoDB with Mongoose
- **Icons**: Lucide React
- **Development**: ESLint, PostCSS

## 🚀 Getting Started

First, install dependencies and run the development server:

```bash
npm install
npm run dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
